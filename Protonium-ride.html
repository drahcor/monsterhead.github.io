Protonium Ride

Description


Protonium Ride est un jeu de type casual game compatible pour les téléphones dans lequel le joueur incarne un petit vaisseau qui doit esquiver différents obstacles et ramasser un maximum d'éléments.
Jouer en faisant glisser votre souris dans le sens vers lequel vous souhaitez vous déplacer pour vous esquiver les obstacles sur ordinateur.
